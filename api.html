開放資料 API

Taiwan Holidays 提供純靜態的 JSON API，您可以直接透過 HTTP GET 請求這些檔案，將最新的台灣假期資料整合進您的系統或應用程式中。不需要申請 API Key，完全免費開放。

Endpoint URL

GET /data/{year}.json

支援的年份 (year): 2017 ~ 2026

💡 使用範例 (JavaScript)

fetch('https://你的網域/data/2026.json')
  .then(response => response.json())
  .then(data => {
    // 找出所有放假日
    const holidays = data.filter(day => day.isHoliday);
    console.log(`2026年總共有 ${holidays.length} 天放假！`);
    
    // 印出詳細資料
    console.log(holidays[0]);
  })
  .catch(error => console.error('Error fetching data:', error));

📦 回傳格式 (JSON)

API 會回傳一個陣列 (Array)，包含該年度每一天的詳細資料：

[
  {
    "date": "20260101",     // 日期 (YYYYMMDD)
    "week": "四",             // 星期幾 (中文單字)
    "isHoliday": true,      // 是否為放假日 (週末或國定假日)
    "description": "開國紀念日" // 節日名稱 (一般週末則為空字串)
  },
  {
    "date": "20260102",
    "week": "五",
    "isHoliday": false,
    "description": ""
  }
]
資料來源備註

所有資料皆整理自中華民國「行政院人事行政總處」公布之政府行政機關辦公日曆表。我們盡力確保資料正確性，但實際放假規定仍應依官方最新公告為準。
